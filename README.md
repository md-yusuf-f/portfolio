# Portfolio

Personal portfolio site for Mohammed Yusuf.

This is a static single-page site built with HTML, Tailwind via CDN, and a small amount of inline JavaScript. It is intended for GitHub Pages, Netlify, or any static hosting platform.

## Highlights

- Hero section with a custom visual style
- Skills and project showcase
- Resume, GitHub, LinkedIn, and contact links
- Responsive layout for desktop and mobile

## Structure

```text
index.html   main site entry point
```

## Local preview

You can open `index.html` directly in the browser, or serve it with a simple static server.

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

This project is suitable for:

- GitHub Pages
- Netlify
- Vercel static hosting

If you publish on GitHub Pages, keep `index.html` at the repository root.

## Featured repositories

The site is intended to showcase projects such as:

- `multi-agent`
- `n8n-webhook-templates`
- `falak-finance-monitor`
